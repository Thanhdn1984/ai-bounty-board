# Why I starred RustChain

I reviewed RustChain's `README.md`, `README_VINTAGE_CPUS.md`, and the bounty-driven issues around Proof of Antiquity. I starred it because the project has a concrete, unusual angle: instead of treating old hardware as waste, it turns vintage CPUs and low-power machines into first-class mining participants. The CPU antiquity/vintage-hardware docs are specific enough to be more than marketing, and the GitHub bounty workflow gives contributors small, verifiable ways to improve docs, tests, and ecosystem reach.

I received RTC compensation for this review.
